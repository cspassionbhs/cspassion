Project Submission Portal

To submit your project to CS Passion, follow these steps carefully:

  1. Create a GitHub account if you don't already have one.
  2. Upload your project (game, website, app, etc.) to a public GitHub repository.
  3. Go to your repository settings and enable GitHub Pages to host your project live.
  4. Test the project link to make sure it loads properly online.
  5. Then, send an email to cspassionbhs@gmail.com including:
    • Your full name
    • Your class
    • Link to your live project (GitHub Pages)
    • Link to your GitHub profile
  6. Once approved, your project will appear on the Projects page.
